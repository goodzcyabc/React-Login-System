# React + Redux Login System

A simple React + Redux application demonstrating login, token authentication (JWT), routing, validation, dynamic styling, server integration, and workflow automation.

---

## Features

- React Router for navigation
- Redux for state management
- Bootstrap for UI
- Form validation using `validator.js`
- Token-based authentication (JWT)
- Dynamic class styling with `classnames`
- Server auto-restart with `nodemon`
- CORS handling via backend or proxy
- Optional MySQL backend integration

---
